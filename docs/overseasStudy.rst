留学
===========

获取banber
~~~~~~~~~~~

**URL**::

    GET      overseas_study/banner

**Parameters**:

* page: 页数，默认值为1
* per_page: 每页记录数, 默认值为20

**Response** `200` ::

    {
        "data":[
            {
                "id":18,
                "jump_type":5,
                "pic_url":"http://pic.hisihi.com/2016-04-14/570f1936bfd9e.jpg",
                "url":"hisihi://university/detailinfo?id=1"
            }
        ],
        "total_count":1
    }
-- end
